Processes an RGB image and returns the pose landmarks on the most prominent person detected.

Parameters

image – An RGB image represented as a numpy ndarray.

Raises

  • RuntimeError – If the underlying graph throws any error.

  • ValueError – If the input image is not three channel RGB.

Returns

A NamedTuple that has two fields describing the landmarks on the most prominate person detected:

  1. ”pose_landmarks” field that contains the pose landmarks.

2) “pose_world_landmarks” field that contains the pose landmarks in real-world 3D coordinates that are in meters with the origin at the center between hips.
