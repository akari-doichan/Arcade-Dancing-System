ddrev.utils.score_utils module¶

ddrev.utils.score_utils.calculate_angle(a: numpy.ndarray[Any, numpy.dtype[float]], b: numpy.ndarray[Any, numpy.dtype[float]], c: numpy.ndarray[Any, numpy.dtype[float]], unit: str = 'radian') → float[source]¶

Calculate angle from 3 coordinates.

Parameters

  • b (npt.NDArray[float]) – A coordinate of a center point.

  • a (npt.NDArray[float]) – Coordinates for other points.

  • c (npt.NDArray[float]) – Coordinates for other points.

  • unit (str) – Angle unit. Defaults to "radian".

Returns

Angles of a, b, c around b.

Return type

float

Examples

>>> import numpy as np
>>> from ddrev.utils import calculate_angle
>>> coords = [
...     np.asarray([0, 1]),
...     np.asarray([0, 0]),
...     np.asarray([1, 0]),
>>> ]
>>> calculate_angle(*coords, unit="degree")
90.0
>>> calculate_angle(*coords)
1.5707963267948966
