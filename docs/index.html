A Novel Arcade dancing System.

QuickStart¶

I recommend you to use these tools to avoid the chaos of the python environment.

  • Pyenv is a python installation manager.

  • Poetry is a packaging and dependency manager.

$ git clone https://github.com/akari-doichan/Arcade-Dancing-System.git
$ cd Arcade-Dancing-System
$ pyenv install 3.8.9
$ pyenv local 3.8.9
$ python -V
Python 3.8.9
$ poetry install
