QuickStart
----------

I recommend you to use these tools to avoid the chaos of the python environment.

- `Pyenv <https://github.com/pyenv/pyenv>`_ is a python installation manager.
- `Poetry <https://python-poetry.org/>`_ is a packaging and dependency manager.

.. code-block:: shell

   $ git clone https://github.com/akari-doichan/Arcade-Dancing-System.git
   $ cd Arcade-Dancing-System
   $ pyenv install 3.8.9
   $ pyenv local 3.8.9
   $ python -V
   Python 3.8.9
   $ poetry install
